# ⭐️ 프로젝트 소개

- 저희 '언더바'는 **독자적인 학과 커뮤니티 플랫폼 개발**을 목표로 합니다.
- 학과 내 다양한 정보들을 담아 **손쉬운 정보 열람**이 가능하며,
